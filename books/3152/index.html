作品简介：

《鬼吹灯》小说作者天下霸唱，讲述了三位当代摸金校尉，利用风水秘术，解读天下大山大川脉搏，寻找失落在大地上的一处处龙楼宝殿，沙漠、雪山、森林、峡谷、急流、草原，鲜为人知的神秘动植物、危机四伏的陷阱，步步惊心，环环紧扣，超越极限与想象力的挑战，在离奇诡异的地下世界中，揭开一层层远古的神秘面纱。 ❀ 影视原著：云南虫谷 / 怒晴湘西 / 河神：鬼水怪谈 / 精绝古城。 ❀ 鬼吹灯系列：第一部 精绝古城 / 第二部 龙岭迷窟 / 第三部 云南虫谷 / 第四部 昆仑神宫 / 第五部 黄皮子坟 / 第六部 南海归墟 / 第七部 怒晴湘西 / 第八部 巫峡棺山 / 鬼吹灯之牧野诡事（全解“鬼吹灯”未解之谜！） / 金棺陵兽（鬼吹灯前传，原名《贼猫》）。 ❀ 摸金校尉系列：摸金校尉之九幽将军。 ❀ 神之三部曲：河神：鬼水怪谈 / 鬼不语之仙墩鬼泣（原名：傩神）/ 无终仙境（原名：殃神 · 鬼家怪谈）。 ❀ 四神斗三妖系列：火神：九河龙蛇。 ❀ 御定六壬同人作品：鬼吹灯之圣泉寻踪 / 鬼吹灯之抚仙毒蛊 / 鬼吹灯之山海妖冢 / 鬼吹灯之湘西疑陵 / 鬼吹灯之镇库狂沙。 ❀ 喜欢鬼吹灯，支持霸唱，购买 正版图书。
