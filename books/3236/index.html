作品简介：

《听雪楼》小说，作者沧月，小说以“听雪楼”的兴衰更迭为背景，讲述了武林“人中龙凤”听雪楼主萧忆情、“血魔之女”舒靖容与热血少年们共同缔造的一段江湖传奇故事。 古装武侠电视剧《听雪楼》由尹涛导演，秦俊杰、袁冰妍领衔主演。 ❀ 电梯直达：壹：血薇 / 贰：风雨 / 叁：神兵阁 / 肆：病 / 伍：火焰鸢尾 / 陆：指间沙 / 柒：护花铃 / 捌：荒原雪 / 玖：铸剑师 / 拾：忘川 · 上 / 拾：忘川 · 下。
