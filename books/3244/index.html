作品简介：

《镜》是沧月奇幻小说系列作品，讲述云荒大陆上的故事。《镜》是一部演绎云荒兴衰史的不朽传奇，讲述了对爱和自由的向往与追求，善与恶的对抗，光明与黑暗的交锋，人的渺小与伟大，生命的脆弱与顽强，信仰、责任、宿命及对命运的抗争。 《镜》系列全部主要作品及外传包括：正传，《双城》《破军》《龙战》《辟天》《神寂》；前传，《神之右手》；外传，《织梦者 · 云荒》《织梦者 · 海的女儿》《六合书·东风破》。《归墟》已合并到《辟天》中，无独立篇章，《六合书 · 讲武堂》未完结，故未收录。 ❀ 电梯直达：镜 · 神之右手 / 镜 · 双城 / 镜 · 破军 / 镜 · 龙战 / 镜 · 辟天 / 镜 · 神寂 / 织梦者 · 云荒 / 织梦者 · 海的女儿 / 六合书 · 东风破。
